# Contracting IPC

Interim Payment Certificates (IPC) Management for Construction Companies.

## Features

- Create and manage Interim Payment Certificates
- Automatic calculation of retention and net amounts
- Workflow support for approval process
- One-click Sales Invoice generation
- Project-based accounting integration

## Installation

```bash
cd ~/frappe-bench
bench get-app https://github.com/your-org/contracting_ipc
bench --site your-site.local install-app contracting_ipc
```

## Usage

1. Create a new IPC document
2. Fill in the contract details and work done amounts
3. Submit for approval through the workflow
4. Once approved, create a Sales Invoice with one click
